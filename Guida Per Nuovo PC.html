Introduzione

Questa guida ti mostrerà come installare e condividere Prompt Perfezionatore, un potente strumento per migliorare i prompt per modelli di linguaggio come ChatGPT, Claude e DeepSeek.

Che tu voglia installarlo sul tuo PC, condividerlo con amici o metterlo online per tester remoti, troverai istruzioni dettagliate e semplici da seguire.

Consiglio: Avere tester umani è fondamentale per migliorare l'applicazione e ricevere feedback prezioso.

Prerequisiti

  • Computer con Windows, macOS o Linux
  • Conoscenze di base sull'uso del terminale/prompt dei comandi
  • Connessione internet

Installazione locale

Segui questi passaggi per installare Prompt Perfezionatore sul tuo PC o quello di un amico.

1

Installa Python

  1. Scarica Python 3.7 o superiore da python.org
  2. Durante l'installazione, assicurati di selezionare "Aggiungi Python al PATH"
  3. Verifica l'installazione aprendo un terminale/prompt e digitando: 
    python --version
2

Scarica il progetto

Con Git
Senza Git
git clone https://github.com/tuonome/prompt-perfezionatore.git
cd prompt-perfezionatore
  1. Crea una cartella chiamata prompt-perfezionatore
  2. Salva al suo interno tutti i file del progetto:
    • prompt_perfezionatore.py (il file principale)
    • requirements.txt
3

Crea l'ambiente virtuale

Apri un terminale/prompt dei comandi nella cartella del progetto:

Windows
macOS/Linux
python -m venv venv
venv\Scripts\activate
python3 -m venv venv
source venv/bin/activate
4

Installa le dipendenze

pip install -r requirements.txt
Consiglio: Questo passaggio potrebbe richiedere diversi minuti, specialmente su connessioni lente o PC meno potenti.
5

Installa il modello italiano per spaCy

python -m spacy download it_core_news_sm
6

Configura la chiave API DeepSeek

  1. Ottieni una chiave API da DeepSeek
  2. Crea un file chiamato .env nella cartella principale del progetto
  3. Aggiungi questa riga al file: 
    DEEPSEEK_API_KEY="la-tua-chiave-api-qui"
    (sostituisci "la-tua-chiave-api-qui" con la tua chiave reale)
Attenzione: Non condividere o esporre pubblicamente la tua chiave API! Trattala come una password.
7

Avvia l'applicazione

python prompt_perfezionatore.py

L'interfaccia web sarà accessibile all'indirizzo: http://127.0.0.1:7860

Successo! Se vedi l'interfaccia di Prompt Perfezionatore, l'installazione è riuscita.

Condivisione in rete locale

Per condividere l'applicazione con amici sulla stessa rete locale (come il tuo Wi-Fi di casa):

  1. Modifica la funzione main() nel file prompt_perfezionatore.py: 
    def main():
    # ... codice esistente ...
    
    # Modifica questa riga:
    interfaccia.launch(share=False, server_name="0.0.0.0")
  2. Avvia l'applicazione: 
    python prompt_perfezionatore.py
  3. Comunica al tuo amico l'indirizzo IP del tuo computer e la porta (solitamente 7860):

    Es. http://192.168.1.5:7860

  4. Il tuo amico potrà accedere all'applicazione dal suo browser usando questo indirizzo
Consiglio: Per scoprire l'indirizzo IP del tuo computer, usa ipconfig (Windows) o ifconfig (macOS/Linux) nel terminale.

Pubblicazione online (per tester remoti)

Ci sono diverse opzioni per rendere disponibile l'app online.

1

Condivisione temporanea con Gradio

Questo metodo è il più semplice ma fornisce un link temporaneo che scade dopo alcune ore:

  1. Modifica la funzione main(): 
    def main():
    # ... codice esistente ...
    
    # Modifica questa riga:
    interfaccia.launch(share=True)  # Imposta share=True
  2. Avvia l'applicazione: 
    python prompt_perfezionatore.py
  3. Verrà generato un link pubblico temporaneo del tipo https://xxx.gradio.app che puoi condividere con i tuoi tester
Attenzione: Il link Gradio scade dopo alcune ore e ha limiti di traffico.
2

Deploy su Hugging Face Spaces (consigliato per tester)

  1. Crea un account su Hugging Face
  2. Vai su Hugging Face Spaces
  3. Clicca su "Create Space"
  4. Seleziona "Gradio" come SDK
  5. Carica i file del progetto (puoi farlo tramite interfaccia web o Git)
  6. Assicurati di configurare la variabile d'ambiente DEEPSEEK_API_KEY nelle impostazioni dello Space
Consiglio: Questa opzione è stabile, professionale e ideale per condividere con più tester.
3

Deploy su un server (per uso prolungato)

Per un servizio più stabile e duraturo, puoi utilizzare un servizio VPS come:

  • DigitalOcean (a partire da $5/mese)
  • Linode (a partire da $5/mese)
  • Render (ha un piano gratuito limitato)

Su questi servizi, segui i passaggi di installazione locale, con l'aggiunta:

def main():
    # ... codice esistente ...
    
    # Per rendere l'app accessibile pubblicamente:
    interfaccia.launch(server_name="0.0.0.0", server_port=8080)

Suggerimenti per i tester

Fornisci queste informazioni ai tuoi tester:

  1. Come accedere all'applicazione (URL)
  2. Esempi di prompt da provare, come:
    • "Parlami dell'energia solare"
    • "Scrivi una email di vendita"
    • "Crea una descrizione di un prodotto per il mio e-commerce"
  3. Specifiche aree su cui desideri feedback, ad esempio:
    • Facilità d'uso dell'interfaccia
    • Qualità dei suggerimenti generati
    • Utilità dell'analisi grammaticale
    • Velocità di risposta
    • Chiarezza della spiegazione delle modifiche
  4. Come segnalare problemi o fornire feedback (email, modulo, etc.)
Consiglio: Crea un modulo Google per raccogliere i feedback in modo strutturato.

Risoluzione dei problemi comuni

Errore "Modulo non trovato"

Assicurati di aver installato tutte le dipendenze:

pip install -r requirements.txt

Errore spaCy

Se appare un errore relativo a spaCy, esegui:

python -m spacy download it_core_news_sm

Errore API DeepSeek

Verifica che la chiave API sia corretta nel file .env

Errore porta in uso

Se la porta 7860 è già in uso, modifica la porta nel codice con:

interfaccia.launch(share=False, server_port=8000)  # Cambia 8000 con una porta disponibile
